Positioning: The Difference between Absolute, Relative and Fixed Positioning


In CSS, the positioning property is used to define an elements spot or 'position' on the web page. There are a few different types of positioning forms that we will differentiate below. This piece looks at three in particular: Relative, Absolute & Fixed Positioning

Relative Positioning is a term that defines an elements position relative to itself. By this definition, this term cast alone will not produce a variation in the elements position. Additional CSS properties must be added to affect the image. For example; left: 100px will drag the element 100 pixels to the right on the images normal position.

Absolute Positioning allows you to place an element exactly where you want it. This element is positioned relative to the parent element that is positioned relatively or absolutely. If no parent exists, the document itself serves as the parent

Fixed Positioning allows an element to remain in a fixed position on a webpage relative to the viewport. Additionally, directional properties such as top:0 are added to set the exact position



  • position: fixed;
  • top:0;
  • right:0:

  • This sets the element to the top right corner of the web page;

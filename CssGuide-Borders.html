What is the Difference Between Margins, Borders and Padding in CSS??


In CSS, the border serves the same function one would serve in the real world, to frame a piece content. An element is the content generally. With CSS, you can adjust several properties of the border, including thickness, style and colour.

Padding is the space between the element and the border, which can be increased or decreased using CSS commands

Margins in CSS refer to the space outside the border. Adjusting the margin controls how this space is used, and gaps between this element and another separate element

Together, these 3 properties form a box of sorts around the content and can be manipulated in many ways to suit the spacial and design needs of your web page


The image shown has the following CSS properties:

img {

  • width:300px;
  • margin-bottom:0%;
  • padding: 10px;
  • border-color: black;
  • border-radius: 20px;
  • border-style: dotted;


    • This CSS defines several properties for the element, which in this case is this image.

    - Width defines the size of the image by how far across the x-axis it traverses.
    - Margin-bottom determines how far from the bottom of the container it rests. In this case 0% means it will sit at the bottom of the designated container for this element
    - The padding size of 10px defines the gap between border and outline of the image, seen above
    - the border properties give clear instructions for the creation of the border

    Keep these distinctions in mind when creating your own web projects
    and play around with the CSS tools to find what suits you best
